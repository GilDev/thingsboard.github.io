Notifications to mobile application<\/h2>\n\n\n\n

We are adding support for sending push notifications to mobile applications. This enhancement includes the ability to load Firebase service account credentials, revamp the dialog for creating notification templates and sending notifications, and allow for the customization of the push notification’s “subject” and “message”. This update aims to enhance mobile application interaction by directly providing timely and relevant notifications to users’ devices.<\/p>\n\n\n\n
